# Photo Portfolio
Kratki opis (1-2 recenice) o cemu je web.
## Stranice
- index.html - Naslovnica.
- projekti.html - Prikaz fotografija. Funkcionira više kao galerija.
- o-meni.html - Sadržaj o autoru (Roko Santini) sa slikom i tekstom.
- kontakt.html - Način za pristupiti u kontakt.
- CSS/stil.css - mapa s CSS kodom
- img - mapa sa slikama
- README.md
## Kako pokrenuti
Otvori index.html u browseru ili koristi Live Server u VS Code.
## Tehnologije
HTML5 i CSS3.
